# Milestone: v1.0 — MVP

**Shipped:** 2026-05-28
**Phases:** 3 | **Plans:** 7 | **Files:** 29 Go files (7,339 LOC)

## Phases

### Phase 1: Foundation — Materializer (3 plans)
Config-driven JetStream consumer materializes events into NATS KV row state.
- MAT-01, MAT-02, MAT-03, MAT-04

### Phase 2: SQL Query Engine + Interfaces (2 plans)
PK-lookup SQL query engine over materialized KV state with NATS request-reply and HTTP endpoints.
- QRY-01, QRY-02, QRY-03, IFC-01, IFC-02

### Phase 3: Packaging + Deployment (2 plans)
Go library API, standalone binary with embedded/external NATS, graceful shutdown.
- IFC-03, DEP-01, DEP-02, DEP-03, DEP-04

## Key Accomplishments

1. Config-driven materializer with YAML/JSON view definitions, composite keys, JSON path column mapping
2. Strict type validation on event ingestion (string/number/boolean/timestamp) with DLQ routing for malformed events
3. PK-lookup SQL query engine using vitess parser supporting =, !=, IN operators with typed JSON results
4. NATS request-reply (natsql.query) and HTTP (POST /api/v1/query) query interfaces
5. Go library facade with three constructors (New, NewWithNATS, NewEmbedded)
6. Cobra CLI with three deployment modes (embedded NATS, external NATS, explicit URL)
7. Graceful shutdown with consumer drain, HTTP server stop, and goroutine leak verification
