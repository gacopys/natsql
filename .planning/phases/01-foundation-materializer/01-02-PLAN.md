      ```go
      type RowMutation struct {
          PK        string            // encoded primary key value
          RowData   map[string]any    // column name → typed value
          StreamSeq uint64            // for _meta injection
          Timestamp time.Time         // event timestamp
      }
      ```

    - `type Mapper struct { viewCfg *ViewConfig; schema *ViewSchema }`

    - `func NewMapper(viewCfg *ViewConfig) (*Mapper, error)`
      - Validates config is non-nil with at least one column
      - Builds schema from viewCfg via `viewCfg.BuildSchema()`
      - Returns Mapper

    - `func (m *Mapper) MapRow(msg jetstream.Msg) (*RowMutation, error)`:
      1. Parse message data as JSON into `map[string]any`
      2. If JSON parse fails → return nil, `ErrMalformedEvent` (D-12: invalid JSON tier)
      3. Extract each column value:
         - Use JSON path from `ColumnConfig.From` (supports dot notation: `user.id`)
         - Walk the map for nested paths: split on `.`, traverse into nested maps
         - If path traversal fails (missing key) → return nil, `ErrMalformedEvent` (D-12: missing key tier)
      4. Validate type per ColumnConfig.Type:
         - `string`: value must be string. Convert numbers via `fmt.Sprint(val)`. Reject other types.
         - `number`: value must be JSON number (float64 in Go). Reject strings/booleans.
         - `boolean`: value must be JSON bool. Reject strings/numbers.
         - `timestamp`: value must be string parseable as RFC3339. Use `time.Parse(time.RFC3339, val)`.
         - Type mismatch → return nil, `ErrMalformedEvent` (D-16: strict, no coercion)
      5. Extract PK value:
         - For each key field, look up the column config, extract value, stringify
         - Join with `KeySeparator` (default `|` per D-04)
      6. Build `RowMutation`:
         - `PK`: joined key values (use `kv.EncodePKValue` for each — but here we just stringify, kv encoding happens at write time)
         - `RowData`: map of column name → typed value
         - `StreamSeq`: msg.Metadata().Sequence.Stream
         - `Timestamp`: msg.Metadata().Timestamp
      7. Return mutation

    - JSON path helper: `func extractNestedField(data map[string]any, path string) (any, error)`
      - Split path on `.`
      - Walk the map tree
      - Return error if intermediate key is missing or not a map

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| JetStream event → Mapper.MapRow | Untrusted JSON payload enters mapping pipeline |
| Mapper → Writer (RowMutation) | Trusted — data already validated by mapper |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | Mapper JSON event parsing | mitigate | Strict schema validation per D-16/D-17. Malformed events return ErrMalformedEvent — never panic. |
| T-02-02 | Denial of Service | Mapper deeply nested JSON | mitigate | Limit JSON depth during path traversal (max 8 levels). Reject events exceeding 1MB payload. |
| T-02-03 | Denial of Service | Writer KV Put failure loop | accept | Writer returns error to caller; processing loop handles retry/Nak. Not writer's responsibility. |
| T-02-04 | Information Disclosure | Row mutation data in memory | accept | All in-process memory. NATS KV encrypts at rest if configured by operator. |
</threat_model>
